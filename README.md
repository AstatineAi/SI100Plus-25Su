### 本地预览

```bash
git clone https://github.com/teafrogsf/SI100Plus.git
cd SI100Plus
npm install -g reveal-md
reveal-md Lecture_01_Python_Intro/Lecture_01_Python_Intro.md -w
```

打开命令行中提示的链接，即可在浏览器中预览课件。
